# Electronic Stethoscope Project

## About the Project  
This project aims to design a digital stethoscope for listening to and analyzing heart and lung sounds(body acoustics) .
By combining traditional stethoscopes with modern digital signal processing techniques.
We can calculate heartbeats (BPM - Beats Per Minute) and respiratory rates (RPM - Respiratory Rate). 
The project has applications in medical diagnostics, telemedicine, and athlete performance monitoring.

---

## Features  
- **Heart Sound Listening and Filtering**: Filters sound signals in the 30-300 Hz range.  
- **Lung Sound Listening and Filtering**: Processes sound signals in the 150-1350 Hz range.  
- **Real-Time Visualization**: Displays raw and filtered signals using MATLAB.  
- **BPM and RPM Calculation**: Calculates heartbeats and respiratory rates using FFT analysis.  

---

## Technical Details  
- **Hardware**:  
  - Stethoscope, microphone, filter circuits  
  - Arduino Uno (for ADC and data transfer)  
- **Software**:  
  - MATLAB (for signal processing and visualization)  
  - Arduino IDE  

---

## Use Cases  
- Medical diagnosis and monitoring  
- Telemedicine applications  
- Athlete performance assessment
